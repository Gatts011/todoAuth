# todoAuth
Postgres CRUD app with Laravel backend and Angular front

Installation

Server:

Run 'composer install; php artisan key:generate'

Dont forget to setup your database in .env (/duplicate, rename .env.example) and 

Run 'php artisan migrate'

then

php artisan serve

Client:

npm i; 

then

npm start
